== Description ==

Create and modify custom status for woocommerce orders.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/deshario-status-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= What does this plugin does ? =

It helps to create and modify woocommerce status.
